# **App Name**: Final Score

## Core Features:

- Event Listing: Display a list of upcoming and ongoing sports events fetched from the Firestore database.
- Real-time Odds Update: Automatically refresh odds from the Firestore database.
- Bet Placement: Allow users to place bets on specific events with specified amounts.
- Return Calculation: Automatically calculate potential returns based on the bet amount and odds, providing clear feedback to the user.
- Admin Panel: Provide an admin interface for managing events, updating odds, and viewing bet data, updating Firebase accordingly.
- Bet Summarization: Provide an AI-powered summarization of a bet. This AI tool considers previous bets, odds, news and team comparisons, and relevant factors before writing the summarization.
- Database seeding: Automatically load example data into the database when the application starts for the first time.

## Style Guidelines:

- Primary color: Deep blue (#1A237E) to convey trust and seriousness in financial activities.
- Background color: Light gray (#EEEEEE) for a clean and professional look.
- Accent color: Vivid orange (#FF5722) to draw attention to important actions.
- Body font: 'Inter', a sans-serif for clear readability and a modern, neutral aesthetic.
- Headline font: 'Space Grotesk', a sans-serif that can deliver a techy feel while providing short, quick-scan readability.
- Use clean, geometric icons to represent different sports and betting options.
- Employ a grid-based layout to organize content effectively, ensuring easy navigation.
